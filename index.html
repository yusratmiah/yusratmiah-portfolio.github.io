Computer Systems: Customizable Unix Shell

Unix Shell Pipelines C

This project develops a customizable Unix shell with robust process management and job control. It executes built-in and external commands, manages foreground/background jobs, and handles signals like SIGINT and SIGTSTP. Supporting job control commands (jobs, fg, bg, kill), it also features I/O redirection, pipelines, and terminal state management for seamless user interaction. The shell is extensible with features like command history and user-configurable prompts.

Java Development: Music Analytics System

Java Data Structures Eclipse IDE

This project implements a Music Analytics System that processes data from the Million Song Database, using hash tables and graphs to efficiently store and analyze artist and track relationships. The system employs an extensible closed hash table for indexing artist names and song titles, dynamically resizing when needed. A graph-based data structure connects artists to their songs, enabling complex queries such as identifying linked records and computing connected components. Users can insert, remove, and retrieve music data through command-line operations, with structured outputs.

C Programming: Comic List Processor

C File I/O Dynamic Arrays

This project implements a dynamic Comic List Processor that reads, manages, and processes comic book data from CSV files. The program loads comics into an in-memory list, supports adding multiple files, and allows users to "buy" comics. This generates an output, which is a receipt with costs, taxes, and totals. Users can display, search, remove, and save comics back into a CSV format. The system efficiently manages memory with dynamic allocation and resizing, preventing memory leaks through careful deallocation, verified using Valgrind.
